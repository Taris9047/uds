1 Heads up Note

This walkthrough is not designed for the ’long’ way. Installing GccEmacs and setting the environment up is not in the scope of this script! If you want to take this ’grandeur’ way, better work with the homebrew compilation scripts!

2 At first, installing ’new’ emacs needs a repo.

sudo add-apt-repository ppa:kellyk/emacs
sudo apt-get update

3 Then install 27.1 by…

 sudo apt-get install emacs27

4 Mods: Doomemacs can be installed with ../../bin/installdoomemacs.sh. But it needs newest git. We also need to add PPA for newest git.

 sudo add-apt-repository ppa:git-core/ppa
 sudo apt-get update
 sudo apt-get install -y git

4.1 Yeah, we are installing git again.

5 Then, the GTK+3 on elementary OS has a big big trouble. We need to set up some environment variable for emacs.

5.1 Add the line below to ~/.bashrc

alias emacs='XLIB_SKIP_ARGB_VISUALS=1 emacs'

5.2 Now we can use emacs normally…
